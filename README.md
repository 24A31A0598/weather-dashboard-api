# Weather Dashboard using REST API

## Project Overview

This project is a real-time Weather Dashboard developed using HTML, CSS, and JavaScript.

The application retrieves live weather information from a public REST API and dynamically displays weather details based on user input.

The project demonstrates asynchronous JavaScript concepts including Fetch API, async/await, JSON parsing, and error handling.

---

## Features

- Search weather by city name
- Retrieve real-time weather data
- Fetch API integration
- Async/Await implementation
- Dynamic DOM updates
- JSON data parsing
- Error handling for invalid requests
- Responsive user interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- REST API
- Fetch API

---

## Weather Metrics Displayed

- Temperature
- Humidity
- Wind Speed
- City Name

---

## Functionalities

### Search
Enter a city name to fetch live weather.

### Fetch Data
Retrieve weather details from REST API.

### Parse JSON
Extract nested weather information.

### Error Handling
Display messages for invalid city names or failed requests.
